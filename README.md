# API de Integração - Dashboard de Estatísticas de Frota

**Introdução**
----
API desenvolvida para a disciplina de projeto integrado de aplicações da Universidade PUC Minas em parceria com a empresa S.S Telemática.

**Tecnologia**
- Express 4.17.1
- bcryptjs 2.4.3
- jsonwebtoken 8.5.1
- Sequelize 5.21.3
- PostgreSQL
